## Architecture Decisions
- **Mock-first backend pattern** — all API calls return mock responses so frontend can be built independently; backend developer replaces mock blocks only
- **Service layer isolation** — UI never calls APIs directly; all requests go through service files (AuthService.js)
- **Hook-based form state** — useLoginForm.js encapsulates validation and state, keeping screens thin
- **Expo SDK 54** — chosen for managed workflow, OTA updates, and broad device support
